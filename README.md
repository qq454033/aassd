#  e-contract

#### 介绍
电子合同签订系统，全程线上签约，秒速签署，支持PC、手机、微信等多终端签署， 随时随地签合同。无需打印、快递合同，线上存储管理，成本节省95%以上。
可根据需要，扩展多模板合同签署。
合同创建完成后，可扩展短信发送给签署方。

#### 软件架构
VS2015+SQL Server,


#### 安装教程

需要部署到IIS上进行测试、调试和使用。

#### 使用说明

1、合同模版pdf+可编辑域制作。
2、根据添加的人员信息自动填充pdf文本域。
3、合同添加后生成短链接（可扩展短信接口将链接以短信形式发送）。
4、打开短链接预览合同，手写签名提交，签署合同
